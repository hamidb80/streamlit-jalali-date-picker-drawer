## Set Env var

### MS Windows
```powershell
$env:JALALI_DATA_PICKER_DEV_SERVER="http://localhost:6001"
```

### Linux
```sh
JALALI_DATA_PICKER_DEV_SERVER="my value"         # local
export JALALI_DATA_PICKER_DEV_SERVER="my value"  # global
```
